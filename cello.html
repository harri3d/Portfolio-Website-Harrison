Burning Cello

In the abandoned concert hall, dead instruments echo old songs. A spark of life stirs.

= currently working on. Video links are private.

Concept - Inspiration

The idea for this project started with a fire assignment for Dynamics class. I created a procedural animation using Maya expressions and set it to 2CELLOS' classical cover of 'Thunderstruck'. The result was unexpectedly promising, and I started to build more of a story in my head.

Painting in Procreate to explore the mood and colors.

Procedural Animation with Houdini MotionFX

I knew that Maya wasn't going to cut it if I wanted a fully procedural setup, so I recreated the cello animation in Houdini - now I could layer subtle noise expressions over each other and create an interface to tweak each parameter.

Matte Painting Photoshop + Nuke

There's something captivating about the eerie atmosphere of abandoned theaters. I knew that I could save a lot of work by using a matte painting, because I only really needed one wide shot to get a sense of the environment. Learning to use projections in Nuke was a challenge. A 3D blockout served as the foundation for my painting. Then I took the render into Photoshop, where I painted layers of detail, decay, and atmosphere.

Painting
Render

By projecting it onto the original geometry in Nuke, I could bring the 2D painting back into 3D space. It doesn't need to be complicated, either. I was pleasantly surprised by how straightforward it was to implement some nice volumetric fog using 3D cards.

Lighting, Color Grading

There was shockingly little dramatically lit cello photography, which necessitated getting creative with reference. A tip if your principal actor is inanimate: you can still use all the portrait lighting tips and techniques, just decide which side of your object is the 'face'.
